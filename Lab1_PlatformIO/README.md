## How to Build and Run (PlatformIO)

1. Open the project in PlatformIO IDE or VS Code with PlatformIO extension
2. Connect your ESP32 board via USB
3. Click the "Upload" button in PlatformIO to compile and flash the program
4. Open Serial Monitor at 115200 baud to view output
